# VideoSearch React Page

Intro

The project uses reactjs to create reusable components that can act similar to youtube, the page uses youtube public api for search terms.

Project component structure

App
    |
    -SearchBar
    |
    -VideoDetail
    |
    -VideoList
        |
        -VideoItem
